Space Emergency Service
========================

This submission is for the 'Udacity HTML5 Game Development Contest', based on HTML5 techniques I learned in this class.

![screenshot 2][2]

---------------------------------------------------
About game
-----------------

Your job is to help other space ships which are in trouble. They can't travel anymore, so you must haul them to the safe zone.
Firsts missions supposed to teach you basics of steering the ship. Then you will learn about space rope and finally you will be tested in emergency situation.

Keys:

 - `Q or mouse button -- starts engine`
 - `W  -- fire space rope`
 - `E  -- release rope`
 - `X  -- restart mission`

---------------------------------------------------
[PLAY][3]
------
Game works the best in Chrome, but also you should be able to play in Firefox. Game doesn't support IE and mobile devices.
